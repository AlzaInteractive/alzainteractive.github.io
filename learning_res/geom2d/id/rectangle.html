Persegi Panjang

Memiliki 4 sisi

Memiliki 4 sudut, masing-masing (90o)

Sisi-sisi berlawanan sejajar dan sama panjang

Keliling

Rumus keliling persegi panjang adalah:

2 x panjang + 2 x lebar

OR

2 x ( panjang + lebar )

Contoh di atas memiliki keliling  2 X (5 + 3)

Luas

Rumus luas persegi panjang adalah:

panjang x lebar

Contoh di atas memiliki luas  5 X 3

Simetri Lipat

Persegi panjang memiliki dua sumbu simetri lipat

Simetri Putar

Persegi panjang memiliki simetri putar tingkat 2

Diagonal

Panjang diagonal sebuah persegi panjang adalah:

√panjang2 + lebar2

(pythagoras formula)

Panjang diagonal dari persegi panjang di atas adalah  √42 + 32
